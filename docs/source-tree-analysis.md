# Source Tree Analysis

## Directory Structure

```
Pro/
├── Assets/                 # [Critical] Store for all media assets
│   ├── BG1.mp4             # Main video background
│   └── *.svg/*.png         # Vector graphics and UI images
├── docs/                   # [Meta] Project documentation
├── index.html              # [Entry Point] Main HTML structure and layout
├── script.js               # [Logic] Vanilla JS for interactions and timer
└── style.css               # [Styling] Global CSS styles
```

## Critical Files

| File         | Purpose                                                                                   |
| :----------- | :---------------------------------------------------------------------------------------- |
| `index.html` | The single page application structure. Contains semantic markup and asset references.     |
| `script.js`  | Handles the countdown timer logic, scroll-triggered animations, and audio unlocking.      |
| `style.css`  | Defines visual styles, including the sticky nav, hero layout, and responsive breakpoints. |

## Key Directories

### Assets/
Contains all heavy media including the 24MB background video and several large SVG composites. This folder is critical for the "Rich Aesthetics" of the site.

### docs/
Contains the documentation generated by the project analysis workflow.
