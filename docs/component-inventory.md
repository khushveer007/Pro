# UI Component Inventory

## Layout Components
| Component             | Description                | Selector                      | Key Features                                 |
| :-------------------- | :------------------------- | :---------------------------- | :------------------------------------------- |
| **Sticky Navigation** | Main navigation bar        | `.sticky-nav`                 | Responsive, Sticky behavior, Mobile toggle   |
| **Hero Wrapper**      | Container for hero visuals | `.hero-scroll-wrapper`        | Sticky viewport, Scroll-triggered animations |
| **Video Background**  | Fullscreen video backdrop  | `.background-container video` | Autoplay, Muted, Fallback interaction        |

## Interactive Elements
| Component            | Description                 | Selector                | Interaction                                  |
| :------------------- | :-------------------------- | :---------------------- | :------------------------------------------- |
| **Countdown Timer**  | 4-unit time display         | `.timer-container`      | Custom JS logic, SVG Frame, realtime updates |
| **Mobile Menu**      | Dropdown for mobile screens | `#mobile-menu-dropdown` | Tap to toggle                                |
| **Scroll Revealers** | Corner decorations          | `.decoration-corner`    | Reveal on scroll                             |
| **Audio Controller** | Background audio            | `.bg-image`             | Unmute logic with user interaction fallback  |

## Decoration Assets
- **Frame Group:** Corner and boundary images (`.frame-group`)
- **Figures:** SVG/PNG decorations that animate on scroll.
